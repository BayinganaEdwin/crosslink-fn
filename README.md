# Crosslink Frontend

## Overview

Crosslink is a web-based application is designed to enhance the quality and structure of internship experiences for university and college students in Rwanda after they have secured placement.

## Getting Started

### Installation

To begin, install the required dependencies using the following command:

```bash
pnpm i
```

# Development Server

After installing the dependencies run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
